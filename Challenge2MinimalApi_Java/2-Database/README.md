# Spring Boot 3 + OpenAPI 3.0

## Requirements

Install Java 21
Install Maven 3.9.6

DONT FORGET TO ADD YOUR ENVIRONMENT VARIABLE (JAVA_HOME, MAVEN_HOME)

## Launch Project

mvn compile    
mvn spring-boot:run

## Swagger

Navigate to: <http://localhost:42069/swagger-ui/index.html>

## Tasks

### Query Generation
